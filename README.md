# Bike Sales Dashboard (Excel)

## Problem Statement
The goal of this project is to analyze bike sales data and create an interactive Excel dashboard that allows stakeholders to quickly understand sales trends, customer demographics, and purchasing behavior.

## Dataset Source
The dataset used in this project was obtained from:
https://github.com/AlexTheAnalyst/Excel-Tutorial/blob/main/Excel%20Project%20Dataset.xlsx

## Approach
- Removed duplicates and handled missing values
- Standardized column names and categories
- Performed exploratory data analysis using pivot tables
- Created calculated fields for deeper insights
- Designed an interactive dashboard using slicers and charts

## Dashboard Features
- Sales depending on customer age and commute (line chart)
- Customer age and income analysis (bar charts)
- Interactive slicers for filtering by demographics

## Results / Insights
- Middle-aged customers generated the highest number of bike purchases
- Higher income groups showed stronger purchasing trends
- Sales seem to be dependent on the customer commute distance

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Slicers)

## How to Use
1. Download the Excel file
2. Open it in Microsoft Excel
3. Use the slicers to filter and explore sales insights
